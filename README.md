# Criando uma Dashboard da Porsche com iA

Este projeto apresenta a criação de uma dashboard executiva inspirada na identidade visual da Porsche, desenvolvida com o apoio de agentes de IA para análise, estruturação e visualização de dados.

A dashboard foi construída em **HTML, CSS e JavaScript**, reunindo filtros interativos, KPIs, gráficos em Canvas e insights automáticos sobre performance de vendas. O objetivo é demonstrar como a inteligência artificial pode acelerar o desenvolvimento de soluções analíticas, desde a leitura dos dados até a criação de uma interface visual sofisticada e funcional.

## Principais recursos

* Dashboard em arquivo único `index.html`
* Visual executivo com estética premium
* Filtros dinâmicos por modelo, ano, cidade, pagamento e período
* KPIs de vendas, receita, ticket médio e liderança por cidade/modelo
* Gráficos renderizados com JavaScript e Canvas
* Insights automáticos gerados a partir dos dados

Este projeto combina **Data Analytics, Front-end e Inteligência Artificial** para transformar uma base de dados em uma experiência visual interativa e estratégica.
